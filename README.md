# 時給計算機

時給・労働時間から、1 日の給与・残業代・月給・年収を計算する静的なページです。残業代には法定割増率 25% を適用します。

## 使い方

ビルドやパッケージのインストールは不要です。

1. リポジトリをクローンまたはダウンロードする。
2. `index.html` をブラウザで開く。

ローカルでファイルを直接開いても動作します。必要なら任意の静的サーバ（例: `npx --yes serve .`）でルートを配信しても構いません。

## ファイル構成

| ファイル | 説明 |
|-----------|------|
| `index.html` | マークアップ |
| `style.css` | スタイル |
| `script.js` | 計算・入力検証 |
| `favicon.svg` | ファビコン（SVG） |

## 計算の前提（概要）

- **通常労働時間・残業時間**は、いずれも **1 日あたり**として入力します。
- **残業代**は、時給 × 1.25 ×（残業の時間＋分÷60）で **1 日分**を表示します。
- **月給**は「（通常分＋残業分）の 1 日あたり合計 × 月の労働日数」です。
- **年収**は「月給 × 12」です。表示は整数円に四捨五入しています。

## 注意

就業規則・36 協定・深夜割増・休日労働など、実際の給与計算では別の扱いになる場合があります。本ツールは **概算用** です。実務の金額は会社の規程や専門家の判断に従ってください。
